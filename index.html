Base URL

Todo lo demás cuelga de aquí:

https://cdn.jsdelivr.net/gh/JahelCuadrado/ExerciseGymGifsDB@main

Cambia @main por @v1.0.0 (u otro tag) cuando quieras congelar la versión.

Endpoints

GET/api/index.json

Metadata global y totales.

GET/api/muscles.json

Lista de grupos musculares.

GET/api/muscles/{muscle}.json

Ejercicios de un grupo.

GET/api/equipment.json

Lista de equipamientos.

GET/api/equipment/{equipment}.json

Ejercicios por equipamiento.

GET/api/bodyparts.json

Lista de partes del cuerpo.

GET/api/bodyparts/{bodyPart}.json

Ejercicios por parte del cuerpo.

GET/api/categories.json

Lista de categorías.

GET/api/categories/{category}.json

Ejercicios por categoría.

GET/api/exercises.json

Todos los ejercicios.

GET/api/exercises/{muscle}/{slug}.json

Detalle individual.

GET/{muscle}/{slug}.gif

El GIF original.

Esquema de un ejercicio

{
  "id": "biceps/barbell-curl",
  "slug": "barbell-curl",
  "name": "Barbell Curl",
  "nameEs": "Curl con barra",
  "muscle": "biceps",
  "bodyPart": "arms",
  "equipment": "barbell",
  "category": "strength",
  "secondaryMuscles": ["forearms"],
  "instructions": [
    "Carga el peso adecuado en la barra y adopta la postura inicial.",
    "Activa el biceps antes de iniciar el movimiento.",
    "..."
  ],
  "file": "biceps/barbell-curl.gif",
  "gifUrl": "https://cdn.jsdelivr.net/gh/JahelCuadrado/ExerciseGymGifsDB@main/biceps/barbell-curl.gif"
}

Valores admitidos

  • bodyPart: arms · legs · chest · back · core · shoulders · cardio
  • equipment: barbell · dumbbell · cable · machine · bodyweight · band · kettlebell · smith · ez-bar · lever · other
  • category: strength · stretching · cardio · plyometrics

Ejemplo de uso

const BASE = "https://cdn.jsdelivr.net/gh/JahelCuadrado/ExerciseGymGifsDB@main";

// Lista de ejercicios de bíceps
const data = await fetch(`${BASE}/api/muscles/biceps.json`).then(r => r.json());

// Mostrar el primero
const ex = data.exercises[0];
console.log(ex.nameEs);           // "Curl con barra"
document.body.innerHTML = `<img src="${ex.gifUrl}" alt="${ex.nameEs}" />`;
